# Viora Curtain Bali - Website

Website profesional untuk Viora Curtain Bali - penyedia solusi gorden berkualitas tinggi di Bali.

## ✨ Fitur

- 🎨 **Desain Responsif** - Tampil sempurna di semua perangkat
- 🏗️ **Interface Modern** - Desain elegan dengan tema warna cokelat yang profesional
- 📱 **Mobile Friendly** - Optimasi penuh untuk mobile devices
- ⚡ **Performance** - Loading cepat dan smooth scrolling
- 📞 **Easy Contact** - Integrasi WhatsApp untuk kemudahan komunikasi

## 📁 Struktur File

```
viora-curtain-bali/
├── index.html      # File HTML utama
├── style.css       # Stylesheet
├── script.js       # JavaScript untuk interaktivitas
└── README.md       # Dokumentasi
```

## 📄 Bagian-Bagian Website

1. **Navigation** - Menu navigasi yang sticky
2. **Hero Section** - Pengenalan produk dengan CTA
3. **Products** - Showcase 3 produk unggulan:
   - Blackout
   - Sheer
   - Roller Blind
4. **About** - Informasi tentang Viora Curtain Bali
5. **Contact** - Informasi kontak dan lokasi
6. **Footer** - Footer dengan social links

## 🚀 Cara Menggunakan

### Lokal
1. Clone repository ini
   ```bash
   git clone https://github.com/aris1208man-pixel/viora-curtain-bali.git
   ```
2. Buka `index.html` di browser Anda

### Deploy ke GitHub Pages
1. Go to repository settings
2. Pilih "Pages" dari sidebar
3. Pilih "main" branch
4. Klik "Save"
5. Website akan accessible di: `https://aris1208man-pixel.github.io/viora-curtain-bali/`

### Deploy ke Netlify
1. Drag & drop folder ke netlify.com
2. Atau connect repository Anda
3. Website akan live dalam hitungan detik
